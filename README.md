# PyGDF - Python Game Development Framework
### Простой фреймворк заточенный на разработку игр на Python, построенный на [Python Program Framework](https://github.com/LukovDev/Python-Program-Framework)

### Описание:
#### Представляет из себя Python Program Framework структуру, в которой есть встроенный python-модуль (ядро/движок) в котором собраны разные скрипты, которые работают воедино, и создают единую среду для разработки игр.</br>
**PyGDF** - Фреймворк предназначенный для разработки игр на Python.</br>
PyGDF объединяет возможности PyGame, OpenGL, OpenAL, PyMunk, Socket, ImGUI, DiscordRPC воедино, чтобы вы могли разрабатывать игры, используя всё это как один мощный инструмент. В ядре движка используется интуитивный и удобный API (похожий на LibGDX/LwJGL), что позволяет создавать мощные и большие проекты.</br>

Этот фреймворк разрабатывался вдохновлёнными фреймворками, такими как ```LibGDX``` и ```LwJGL```</br>

Не путать с библиотекой NVIDIA PyGDF для GPU-вычислений!

### Статус проекта:

На данный момент, проект заморожен (скорее всего навсегда).</br>
Причина: Ограничение производительности языка Python.</br>
Следствие: Переход на языки C/C++ и разработка прямого наследника - [RAW Engine](https://github.com/LukovDev/RAW-Engine/).</br>
Ядро движка **RAW Engine** ([RAWE](https://github.com/LukovDev/RAW-Engine/)) - Это новое поколение ядра **GDF** (из этого фреймворка), но с существенными улучшениями и переходом на чистый язык Си.</br>

Этот проект приостановлен на этапе перехода OpenGL на новую версию, без использования устарелого функционала.</br>
Последнее что разрабатывалось - Переход пакетной отрисовки спрайтов на новый OpenGL. Большинство графического функционала было перенесено.</br>
Последние изменения и статус был зафиксирован в ветке [development-alpha](https://github.com/LukovDev/PyGDF/tree/development-alpha).</br>
Прогресс остановлен и находится в "подвешенном" состоянии. Работа ядра и кода нестабильна и требует анализа и доработки.

Прошу прощения, что бросил проект на подвешенном  состоянии.
